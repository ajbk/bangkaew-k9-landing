# Bangkaew K9 Landing Page

Landing page starter for **Bangkaew K9 — AI Cyber Watchdog for Thai SMEs**.

This project is built as a **Next.js + TypeScript + Tailwind CSS** single-page landing site so you can continue in Codex, Cursor, VS Code, or deploy to Vercel.

## Core positioning

**Bangkaew K9** is an AI Cybersecurity Platform + Edge Appliance for Thai SMEs.

Tagline:

> ดมไว เห่าทัน เฝ้าบ้านอย่างใส่ใจ

Hero:

> Own Your Cyber Watchdog.

Subhero:

> AI-powered cybersecurity platform and edge appliance for Thai SMEs.

## Sections included

1. Hero + CTA
2. The Cyber Watchdog Platform
3. Meet K9 Edge Pro
4. How Bangkaew K9 Works
5. Built for Thai SMEs
6. Why Bangkaew K9
7. Final CTA
8. Footer

## Run locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Deploy to Vercel

```bash
npm install -g vercel
vercel
```

or push to GitHub and import the repository into Vercel.

## Files to edit first

- `app/page.tsx` — content and layout
- `app/globals.css` — visual style, device mockup, dashboard mockup
- `app/layout.tsx` — metadata / SEO title
